# Nano Banana Pro: Community Fan Build (v2.2.0)

Это неофициальная фанатская сборка (Community Edition), созданная для оптимизации работы 
фреймворка Nano Banana Pro с локальными графическими процессорами RTX 30-й серии.

### Основные функции сборки:
* **Локальная интеграция:** Полная поддержка формата `.safetensors` для работы в офлайн-режиме.
* **Оптимизация VRAM:** Настроено под 12 ГБ видеопамяти (тестировалось на RTX 3060).
* **Расширенные слоты:** Поддержка до 30 слотов для пользовательских модификаций и стилей.
* **Улучшенный рендеринг:** Интеграция с CyberRealistic для повышения детализации в режиме Inpaint.

### Как использовать:
1. Клонируйте репозиторий в папку вашего Stable Diffusion Forge.
2. Убедитесь, что файлы `config.json` и `init_index.json` находятся в корневой директории.
3. Поместите ваши авторские LoRA в соответствующие слоты (17-18).
4. Запустите систему в режиме `--offline` для максимальной производительности.
